# __<center><font color="#006666">指针</font></center>__
## __<font color="#006666">指针的长度</font>__
- 在x64中指针都是8byte

## __<font color="#006666">*号的含义</font>__
- 定义的时候(前面有类型)，表示后面的变量是指针
- 使用的时候，表示取值(取指针指向的内存空间的值)
- int* 和 char* 的步长是不一样的

## __<font color="#006666">xor</font>__
- 0 xor 任何数 = 任何数
- 1 xor 任何数 = 任何数取反
- 任何数 xor 自己 = 把自己置0
